# connect4-node-react
Connect 4 game implemented with React with NodeJS

### Pre-requirements
- node & npm

### To get started

1. `cd` into your project directory
2. run `npm install`
3. run `node server.js` or 'nodejs server.js' (ubuntu)
4. visit <http://localhost:3000/>
